# ParminCloud S3 Object Storage Browser

## Powered by

* pnpm
* React
* Nuxt.JS
* Chakra UI
* TypeScript
* react-icons
* JavaScript S3 Client

## Feature

* Directory/Sub-Directory support
* Browse files (limited to first 100 items) [See TODO list]
* Object actions (Upload, Download, Remove, etc.)
* Dark/Light Mode :)
